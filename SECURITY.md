# Security Policy

## Reporting a Vulnerability

Please do **not** open a public GitHub issue for security vulnerabilities.

Report them privately through GitHub's **[Security Advisories](https://github.com/wannabexaker/Cypher/security/advisories/new)**
(Security → Advisories → Report a vulnerability).

Include reproduction steps and the affected route/component where possible. You can
expect an initial acknowledgement within a reasonable timeframe.

## Scope

Cypher handles guest voting and anti-fraud, file uploads, and authentication.
Reports about the vote pipeline, upload handling, authorization, or session
handling are especially welcome.
